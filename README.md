# did:ethr intro

An interactive slide-deck that explains how `did:ethr` works, with a story-driven approach and a focus on the underlying
mechanics of DID resolution on Ethereum.

## Live version

Run the slides live at [https://mirceanis.xyz/ethr-intro](https://mirceanis.xyz/ethr-intro).

You might want to use [https://mirceanis.xyz/ethr-manager](https://mirceanis.xyz/ethr-manager) to follow along with the
story and see how the DID document changes as transactions are sent to the registry contract.

## To run locally:

```bash
pnpm install
pnpm run dev
```

Then open http://localhost:5173 in your browser.
